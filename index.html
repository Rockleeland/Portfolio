Web Development
MadPass

This application allows a user to quickly and easily save and store login and password information for a variety of websites, accounts, apps, etc. We used Node.js, Express.js, MongoDB / Mongoose, Crypto npm module to encrypt data, encryption algorithm AES (Advanced Encryption System) with a 256 bit encryption key, and Bcrypt. I worked on both the front and back end making sure the layout and UI was simple and clean, and assuring that redirection of the pages along with login and password management were successful.
